# React Native & Watch Implementation Notes

## Future Mobile Application
- Port Core Logic: Share logic between React (Web) and React Native (Mobile).
- UI Components: Utilize `View`, `Text`, `TouchableOpacity` from `react-native`.
- State Management: Consider `Context API` or `Zustand` for cross-platform state.

## Future Apple Watch / Android Wear Application
- Complications: Display the current hole and score.
- Input: Large buttons for score entry.
- Haptics: Vibrate when a score is entered or a hole is completed.
- WatchOS/WearOS specific layouts.
